# Virtual Dressing Web Frontend

This is the Next.js 14 frontend for the Virtual Try-On Avatar Platform.

## Features
- Auth (email/OAuth)
- Image upload (Uppy/Dropzone)
- Avatar preview
- Wardrobe grid
- Try-on viewer
- Save/share looks

## Getting Started
- Install dependencies: `npm install`
- Run dev server: `npm run dev`
